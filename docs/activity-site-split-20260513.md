# Activity Site Split Update

Date: 2026-05-13

## Scope

- `#/activities` is the activity hub.
- The hub links to four DOM detail routes:
  - `#/forest-school`
  - `#/field-school`
  - `#/regional`
  - `#/waribashi`

## Route Behavior

- `#/forest-school` and `#/field-school` show activity-specific overview content plus matching recruitment snapshots.
- `#/regional` shows the regional-block guide.
- `#/waribashi` shows the standing initiative route and keeps the official JUON page as the final-detail link.

## Contract Change

These four routes are no longer mockup-only provisional routes. Their old mockup PNGs remain reference assets only.
